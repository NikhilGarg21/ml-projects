# Machine Learning Practice Repository

This repository contains my Machine Learning practice projects built while learning and experimenting with different datasets, preprocessing techniques, and models. It focuses on strengthening core ML concepts through hands-on implementation.

---

## Overview

This repository includes:
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering techniques
- Supervised learning models
- Model evaluation and performance metrics
- Practice experiments on different datasets
